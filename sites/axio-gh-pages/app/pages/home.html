Development easy.

About

AXIO.JS is Javascript framework designed to be extremely simple, flexible, powerful and fast.

It represents everything as a lot of Entities exchanging Events.

It intended to create complex web-client, mobile or server-side solutions.

It also includes set of basic UI components, extensions and plugins to integrate with third party services and libraries.

Features

easy to use
1) Define entity/property types you need 2) and then create entities.

flexible
Entity types may extends each other or mixed by its properties.

loose coupled
Event-driven, property binding with expression, reactive.

asynchronous & error-aware
Async properties load with built-in error handling.

incremental code loading
Code bundles loaded at once on demand

independent
No any external dependencies,but easy to integrate with any

extremely fast
Own well-optimized and powerful infrastructure, pure Javascript

multi-tier
Successfully used both on client and server side.

modern web oriented
HTML5/CSS3, many HTML5 APIs, no legacy.

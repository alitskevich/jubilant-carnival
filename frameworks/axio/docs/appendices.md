Abbreviations
=============

There is list of nice abbreviations widely used in code.

+ **a, arr** : array
+ **args** : arguments
+ **cb** : callback
+ **ctx** : context
+ **cfg** : configuration
+ **cstr** : constructor
+ **d** : I. data II. date
+ **e, err** : error
+ **ev** : event
+ **ex** : exception
+ **fn** : function
+ **i** : index
+ **n** : name
+ **o, obj** : object
+ **p** : pointer, position
+ **r** : result
+ **s** : string
+ **T** : reference to `this`
+ **v, val** : value
